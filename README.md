# MP Stock Tracker v3

Google Apps Script для автоматического сбора остатков с маркетплейсов OZON и Wildberries, выгрузки остатков из МойСклад, синхронизации кредитов с управленческим балансом. Уведомления через Telegram.

## Модули

| Модуль | Описание |
|--------|----------|
| 📦 Остатки маркеты | Сбор остатков OZON / WB по всем кабинетам |
| 🏪 Остатки МС | Выгрузка остатков из МойСклад с себестоимостью |
| 💳 Кредиты | Синхронизация остатков по кредитам в управленческий баланс |

## Быстрый старт

### 1. Создайте Google Таблицу

Откройте Google Sheets → Extensions → Apps Script → вставьте содержимое файла `mp_stock_tracker_v3.gs`.

### 2. Сохраните и обновите таблицу

После сохранения скрипта обновите страницу таблицы. В меню появится пункт **Автоматизация**.

### 3. Заполните Панель управления

Меню создаст лист **Панель управления** автоматически. Заполните:

| Ячейка | Что вписать |
|--------|-------------|
| B2 | Токен Telegram-бота (от @BotFather) |
| B3 | Chat ID для уведомлений (несколько — через `;`) |
| B5 | API-токен МойСклад (Настройки → Доступ по API) |

### 4. Добавьте кабинеты маркетплейсов

На листе **Панель управления** начиная со строки 9:

| Столбец | OZON | WB |
|---------|------|----|
| A | `OZON` | `WB` |
| B | Название ИП | Название ИП |
| C | Client ID | — |
| D | API Key | — |
| E | — | Token |
| F | Имя листа для данных | Имя листа для данных |
| G | `Да` / `Нет` | `Да` / `Нет` |

## Меню Автоматизация

### 📦 Остатки маркеты

- **Обновить все кабинеты** — запускает сбор остатков по всем активным кабинетам
- **Выбрать кабинеты...** — выбрать нужные кабинеты из списка
- **Один кабинет...** — запуск для одного кабинета
- **Только OZON / Только WB** — фильтр по маркетплейсу
- **Добавить кабинет** — добавить новый кабинет через диалог
- **Проверить низкие остатки** — уведомление в Telegram о позициях ≤ 5 шт.
- **Проверка API** — проверить доступность API всех кабинетов
- **Обновить дашборд** — сводный лист по всем кабинетам
- **Экспорт дашборда в CSV** — сохранить дашборд в Google Drive
- **Настроить автозапуск...** — триггер по расписанию (каждые N часов)
- **Удалить все триггеры** — удалить все автозапуски

### 🏪 Остатки МС

- **Загрузить остатки сейчас** — выгрузить остатки из МойСклад в лист `Остатки МС`
- **Включить ежедневный автозапуск** — запускать каждый день в ~7:00
- **Отключить ежедневный автозапуск**

### 💳 Кредиты

- **Предпросмотр данных** — показать что будет записано без изменений
- **Записать в последний столбец** — синхронизировать в последний заполненный столбец Управленческого баланса
- **Записать в конкретный столбец...** — выбрать столбец вручную (A, B, C…)

### ⚙️ Сервис

- **Очистка старых данных...** — удалить строки старше N дней
- **Миграция из старых листов** — перенести конфигурацию из листов `Технический Озон` / `Технический ВБ`

## Листы таблицы

| Лист | Назначение |
|------|------------|
| Панель управления | Токены, список кабинетов |
| Дашборд | Сводка по всем кабинетам |
| Лог | История запусков |
| Остатки МС | Остатки из МойСклад |
| Остатки по кластерам ИП* | Данные OZON (имя задаётся в столбце F) |
| Остатки ВБ ИП* | Данные WB (имя задаётся в столбце F) |
| Кредиты Import | Источник данных по кредитам |
| Управленческий баланс | Целевой лист для записи кредитов |

## Адаптация модуля Кредиты

В скрипте есть константа `CREDIT_MAP` — маппинг между названиями кредитов в листе `Кредиты Import` и строками в листе `Управленческий баланс`. Отредактируйте её под свою структуру данных.

Ожидаемые заголовки в листе `Кредиты Import`:
- `Основной долг` — сумма долга
- `Кредит` — название кредита
- `Оплачено` — признак оплаты (`Да` / `Нет`)

## Требования

- Google Workspace или личный Google-аккаунт
- Доступ к Google Apps Script
- Telegram-бот (для уведомлений)
- API-ключи OZON Seller, WB Statistics, МойСклад (по необходимости)
