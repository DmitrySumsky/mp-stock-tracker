# Changelog

## [3.0.0] - 2026-03-30

### Added
- Интеграция МойСклад: выгрузка остатков с себестоимостью в лист «Остатки МС»
- Поле API-токена МойСклад в Панели управления (B5)
- Модуль «Кредиты»: синхронизация остатков по кредитам в Управленческий баланс
- Мульти-чат Telegram (несколько chat_id через `;`)
- Дашборд с автообновлением и экспортом в CSV
- Разбивка монолита на модули (`src/00_constants.gs` … `src/99_menu.gs`)

### Changed
- Меню переименовано: «📦 Остатки МП и другие скрипты» → «Автоматизация»
- Подменю разделены на три независимых модуля: Остатки маркеты / Остатки МС / Кредиты
- `TABLE_START_ROW` сдвинут с 7 на 9 (добавлен блок МойСклад в строках 4-5)

### Security
- Удалены захардкоженные Telegram-токен и chat ID из функции миграции

## [2.x] — Предыдущие версии

Монолитный скрипт с поддержкой OZON и WB, без МойСклад и модуля кредитов.
